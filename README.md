
# A simple CRUD/DB Testing Area

### Usage
```unix
  cd/"project-folder"
```

In Terminal
```unix
  bundle install
```

Subjects
* id (primary key)
* name
* position
* visible (boolean)


Pages
* id (primary key)
* subject_id
* name
* permalink
* position
* visible (boolean)

Sections
* id (primary key)
* visible (boolean)
* content_type
* content
